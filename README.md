# eden-query

> @elysiajs/eden + @tanstack/query integration

## Getting Started

1. Clone the repository and change into the correct directory.

```sh
git clone https://github.com/ap0nia/eden-query.git
cd eden-query
```

2. Install dependencies.

```sh
pnpm install
```
